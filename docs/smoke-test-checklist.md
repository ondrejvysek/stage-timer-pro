# Smoke Test Checklist

Run this before merging core behavior changes.

```text
- [ ] npm install

- [ ] open moderator UI
- [ ] open presenter UI
- [ ] start countdown

- [ ] resume countdown
- [ ] reset countdown
- [ ] switch display mode
- [ ] show and hide message
- [ ] restart service while paused
- [ ] restart service while running
- [ ] check logs for errors
```
